# Comment ça marche ?
1. L’IA regarde la case actuelle
2. Si c’est un mur → stop
3. Si c’est l’objectif → gagné
4. Sinon → elle marque la case comme visitée
5. Puis elle appelle la même fonction dans les 4 directions

# L’IA va :
• Explorer
• Cochez les cases
• Eviter les murs
• Eviter les boucles
• Trouver l’objectif si possible
